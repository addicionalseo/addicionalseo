Per què importa

La web corporativa no hauria de ser només una presentació.

Quan la proposta de valor no és clara, quan els serveis estan mal ordenats o quan la web depèn massa de plantilles, la captació es ressent.

Treballar una web corporativa amb criteri vol dir ordenar missatge, estructura, jerarquia, SEO i desenvolupament.
