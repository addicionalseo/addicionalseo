FAQ

Preguntes habituals sobre disseny web per a pimes i particulars.

Quant costa una web per a una pime?

Una web corporativa professional per a una pime sol estar entre 1.200 i 2.500 €. Depèn del número de pàgines, la complexitat i el disseny. Ho valorem cas per cas sense compromís.

Necessito una web cara per posicionar a Google?

No. El que necessites és una web ben construïda, amb bon SEO on-page i contingut rellevant. El preu no determina el posicionament, la qualitat tècnica i el contingut sí.

Podeu fer la web si no tinc contingut preparat?

Sí. T'ajudem a definir l'estructura, els textos i el missatge comercial. Molts clients ens passen idees i nosaltres fem la redacció optimitzada per SEO.

Quant triga a estar llesta?

Entre 3 i 5 setmanes per a una web corporativa estàndard. Si teniu pressa, ho podem accelerar. Si necessiteu una landing page senzilla, pot estar en menys de 2 setmanes.

Feu webs per a particulars?

Sí. Si tens una idea de negoci, un projecte personal o una marca que vols crear, t'ajudem a fer-ho de forma professional amb un pressupost ajustat. No cal tenir una empresa constituïda per tenir una web ben feta.
